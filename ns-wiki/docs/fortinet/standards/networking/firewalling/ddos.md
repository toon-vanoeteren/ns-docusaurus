# DDoS  

> Any application connected to the Internet can be subject to a distributed denial of service (DDoS) attack. Azure provides multiple plans to protect your networks seamlessly.

## Plans  

Azure provides 2 possible plans for DDoS protections:

|SKU |Info |
|:---|:---|
|DDoS IP Protection |This protection is activated on select IPs and provides protection against most L3/L4 attacks. |
|DDoS Network Protection |This protection is activated on subnet level and provides more protections compared to the IP plan. |

:::info
An in-depth comparison can be found [here](https://learn.microsoft.com/en-us/azure/ddos-protection/ddos-protection-sku-comparison?source=recommendations#tiers)
:::
