# Front Door

## Overview

Azure Front Door provides a Content Delivery Network (CDN) for application and APIs of the customers. 

### Key features

* **High Availability**  
Front Door is a global loadbalancer providing high availability for applications in Azure, on-premise or other cloud providers.

* **Low latency**  
Front Door utilizes hundreds of Microsoft's edge networks for delivering the application.

* **Secure**  
Front Door is natively protected by L3-4 DDoS protection. For web applications protection, an `Application Gateway` can be integrated.
